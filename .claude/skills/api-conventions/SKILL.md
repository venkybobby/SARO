# API Conventions Skill

## Trigger Conditions
Activate for changes to `main.py`, `routers/*.py`, `schemas.py`, or any file using `APIRouter`, `@app.get`, `@app.post`.

## Route Rules

```python
# ALL handlers must be async
@router.post("/scans", response_model=ScanResult)
async def create_scan(payload: ScanRequest, db: AsyncSession = Depends(get_db)):
    ...

# URL prefix — always /api/v1/
router = APIRouter(prefix="/api/v1/scans", tags=["scans"])

# Never use blocking calls in async handlers
# WRONG: time.sleep(1), requests.get(url), df.to_csv()
# RIGHT: await asyncio.sleep(1), await httpx.get(url), await run_in_executor(...)
```

## Pydantic v2 Patterns

```python
from pydantic import BaseModel, Field, model_validator

class ScanRequest(BaseModel):
    model_config = ConfigDict(str_strip_whitespace=True, frozen=True)

    prompt: str = Field(..., min_length=1, max_length=32_000)
    raw_output: str = Field(..., min_length=1, max_length=128_000)
    vertical: Literal["finance", "healthcare", "technology", "government"]

    @model_validator(mode="after")
    def validate_non_empty(self) -> "ScanRequest":
        if not self.prompt.strip():
            raise ValueError("prompt cannot be blank")
        return self
```

## Standard Error Response

```python
class ErrorDetail(BaseModel):
    code: str        # e.g. "SCAN_FAILED", "INVALID_VERTICAL"
    message: str
    request_id: str  # UUID from X-Request-ID header

# Use HTTPException with detail=ErrorDetail(...).model_dump()
raise HTTPException(status_code=422, detail=ErrorDetail(
    code="INVALID_VERTICAL",
    message="vertical must be one of: finance, healthcare, technology, government",
    request_id=request_id
).model_dump())
```

## JWT Middleware

- All routes except `GET /health` and `POST /api/v1/auth/token` require `Authorization: Bearer <jwt>`.
- Use `Depends(get_current_user)` — never decode JWT inline in a handler.
- Token expiry: access = 15 min, refresh = 7 days.
- Algorithm: HS256 minimum; prefer RS256 in production.

## Required Headers

```
X-Request-ID: UUID4   (generated by middleware if absent)
X-SARO-Version: 8.0.0
```

## Health Endpoint

```python
@app.get("/health")
async def health():
    return {"status": "ok", "version": "8.0.0"}
```

## What's Forbidden

- `requests` library in any router — use `httpx.AsyncClient`.
- `print()` for logging — use `structlog.get_logger()`.
- Bare `except Exception` without re-raise or explicit structured log.
- Synchronous SQLAlchemy sessions in async handlers.
